# recolearn

Uniwersalna platforma internetowa do powtórek, quizów i nauki z wykorzystaniem fiszek.  
Pozwala uczniom i nauczycielom efektywnie utrwalać wiedzę, organizować własne zestawy i śledzić postępy.

---

## Najważniejsze cechy projektu

- Tworzenie i zarządzanie własnymi zestawami fiszek/pytań
- Inteligentny system powtórek (krzywa zapominania)
- Prosta rejestracja, logowanie i resetowanie hasła
- Panel administratora (Django Admin) – szybkie zarządzanie zasobami
- Nowoczesny, responsywny design
- Obsługa plików statycznych i uploadowanych grafik
